# AniScope v3.0

AniScope is a personal anime recommendation and tracking PWA.

## Run locally
Open `index.html` with VS Code Live Server.

## Deploy
Commit and push this folder to the GitHub `main` branch. Cloudflare Pages deploys automatically.

## Structure
- `assets/artwork` — full anime posters
- `assets/thumbnails` — list-card artwork
- `assets/icons` — PWA and brand icons
- `css` — styling
- `js` — application logic
- `data` — anime data
- `docs` — roadmap, design system, changelog
